# 場面名
引き渡し前に確認すべきことがわからない

## いつ起きる？
完成書類が揃った直後／引渡し当日の数日前／施主への鍵・操作説明の準備時／元請けへの最終提出直前

## 引き渡し前とは（先に押さえる）
- **001〜029 の全工程の最終結実点**（着工〜検査〜是正〜竣工図〜完成書類の総まとめ）
- 元請け・施主に「**完成しました、お渡しします**」を正式に伝える場
- ここで抜けが出ると **施主クレーム＋是正命令＋信用ガタ落ち**
- 鉄則は **「全工程完了 ＋ 現地最終一周 ＋ 施主デモ準備 ＋ 書類／鍵 一式」**
- **新人のうちは臆病なくらいがちょうどいい**

## 若手担当者が止まる理由
- 最終確認の **範囲・順序**が分からない（もう全部見た？それともデモだけ？）
- 完成書類と as-built 図面の **整合性**で迷う
- 現地と図面の**最後のズレ**（是正後の反映漏れ）に気づいて止まる
- **施主デモ**（操作説明・取説読込）の準備が間に合わない
- **鍵・リモコン・パスコード**の引渡しリストを作ったことがない
- 元請け・施主への**立会連絡のタイミング**で迷う
- 一人で抱え込んで「**これで本当に大丈夫か**」が確信できない

## 怒られるポイント
- 最終確認飛ばしで**当日に不具合発覚** → 「**検査の時に確認してなかったのか**」
- **書類不備**（校正証明書・免状コピー欠け） → 「**工事として認められない**」
- **器具汚れ・ゴミ残し** → 「**プロとして恥ずかしい**」
- 施主が触って動かず「**説明聞いてない**」 → デモ準備不足
- **是正後の図面反映漏れ** → 引き渡し直前に「現地と違う」発覚
- 鍵・リモコン・パスコードの**引渡しリスト無し** → 後日「これどう使うの？」攻撃
- 一人で全部やって**疲労からのケアレスミス**

## 本当は確認すべきこと
1. **全前工程 100% 完了**
   - 020〜029（結線・導通・絶縁・点灯・動作・消防・施主・是正・竣工図・完成書類）
   - 異常・指摘は**すべて是正済み**
2. **現地最終一周チェック**
   - 全回路通電確認（**異常音・発熱・チカチカ**なし）
   - 器具清掃・**プレート水平・ビス向き統一**
   - コンセント・スイッチ動作
   - **非常灯・誘導灯点灯**（停電時バッテリー含む）
   - **電気錠・インターホン・スマート設備**の設定完了
3. **施主デモ準備**
   - 操作説明用取説を**施主用に揃える**
   - **デモ練習**：「ここはこう使います」を声に出して
   - 想定質問への回答準備
4. **書類・鍵・アクセス**
   - **完成書類一式**を提出形式（原本／コピー／PDF）で準備
   - **鍵・リモコン・パスコード**の引渡しリスト作成
   - 保証期間・連絡先の明示
5. **安全・連絡**
   - 元請け・施主に「**引渡し準備完了、立会お願いします**」
   - 周囲周知＆**最終無電圧確認**（必要箇所）
6. **記録**
   - **最終写真**（完成形・全箇所）
   - チェックリスト記入
   - 異常時は**即報告・即是正**

## まず見る図面・資料
1. **元請けの引渡し要領書・提出書類一覧表・特記仕様書** ← まずこれ
2. **as-built 竣工図一式 ＋ 完成書類一式**（029 で揃えたもの）
3. 前工程全記録（試験成績書・**是正リスト・施工写真**）
4. **機器取説・保証書・操作マニュアル**（施主デモ用）
5. 総合図・消防同意書・設計変更指示書（連動設備の最終確認）
6. 鍵・リモコン・パスコードの一覧表

## 誰に聞く？
- **元請け現場監督**（引渡し段取り・立会・最終承認）
- 自社の先輩現場代理人・施工管理者
- 自社職長（清掃・美観・最終一周の社内ルール）
- メーカー（**特殊機器のデモ・操作説明**）
- 消防設備業者・弱電業者（連動先の最終確認）
- 施主・管理会社（**鍵・パスコード・引渡し時間**）

## 聞き方テンプレ
「**引渡しの段取り・所要時間・立会者**を教えてください」
「**完成書類**は最終チェック OK ですか？提出形式・部数は？」
「**鍵・リモコン・パスコード**の引渡しリスト、社内フォーマットありますか？」
「**施主デモ**の練習、立ち会ってもらえますか？」
「**是正後の図面反映**、最終チェックお願いします」
「保証期間・**緊急時の連絡先**はどう案内しますか？」

## 職長への指示テンプレ
「引渡し前に**現地を一周**、全器具動作・美観・清掃を再確認」
「**異常音・発熱・チカチカ**を全回路で見逃さない」
「**非常灯・誘導灯のバッテリー点灯**まで確認してください」
「**完成書類・鍵・リモコン・パスコード**の引渡しリストを作成」
「**施主デモ練習**を声に出してリハーサル」
「**最終写真**（完成形）を全箇所で残す」
「**疲労時は絶対一人で確認しない**、ペアで」

## 事前チェック3つ
1. **020〜029 の OK 記録**＋**現地最終一周（機能・美観・設定）**が完了しているか
2. **完成書類一式 ＋ 鍵／リモコン／パスコード引渡しリスト ＋ 施主デモ練習**が揃ったか
3. **元請け・施主への立会連絡＋是正後の図面反映＋最終写真**の段取りが組めているか

> 補足：引渡しは **001〜029 の全工程の結実点**。**抜けが出ると全部に泥を塗る**。
> 補足：**新人のうちは臆病なくらいがちょうどいい**。「**大丈夫だろう**」が事故の元。
> 補足：**鍵・リモコン・パスコード**は意外と忘れやすい。引渡しリストを必ず作る。
> 補足：**疲労時は一人で完結しない**。ペアで一周＋上司チェックで仕上げる。

## NG行動
- 020〜029 の OK 記録を再確認せず引渡し当日を迎える
- 現地最終一周（機能・美観・設定）を省略する
- 是正後の図面反映を確認しない
- 施主デモの練習をせず本番に臨む
- 鍵・リモコン・パスコードの引渡しリストを作らない
- 完成書類の控えを保管しない
- 立会連絡を直前まで放置する
- 疲労時に一人で最終確認する
- 異常を「様子見」で放置する
