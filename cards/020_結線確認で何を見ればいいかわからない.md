# 場面名
結線確認で何を見ればいいかわからない

## いつ起きる？
器具・盤・制御機器の結線作業中／結線完了後の自主検査時／絶縁・通電試験の直前／改修・既設機器の結線変更時

## 結線確認とは（先に押さえる）
- **配線・端子接続が図面通り・仕様通りに繋がっているか**を確認する作業
- **無電圧確認 → 導通 → 端子・色・極性 → 締付 → 写真**の手順が基本
- ここでミスると **絶縁不良・誤動作・逆相・短絡・火災**につながる
- 「**疲れた時の自分を信用しない**」が鉄則

## 若手担当者が止まる理由
- **単線図だけ**見て複線図のイメージができない
- 図面と現場のズレ（既設・他業種干渉）に気づいて止まる
- **端子番号・マーキング**が無く、外す時の復旧が不安
- 疲労時の自信喪失（**17時過ぎ**のケアレスミス問題）
- テスターの**導通モード**の使い方に慣れていない
- 「大丈夫だろう」で確認を飛ばしてしまう
- 制御線・リレー回路など複雑な結線で頭が真っ白になる

## 怒られるポイント
- **逆相・色間違い・緩み**で機器が動かない／消防設備不動作
- 図面と実際の**終端位置が逆** → 全回路が機能不全
- **テスター未確認**で進めて再工事
- 制御線を外す時に**写真を撮らずバラす** → 復旧不能で大目玉
- 締め付け不足で**接触不良** → 発熱・火災リスク
- 設計変更が反映されていない古い図面で結線
- 「説明したのに施工しない」 → 「**甘やかさない**」と厳しく指導

## 本当は確認すべきこと（標準手順）
1. **安全確保（最優先）**
   - 対象回路の**ブレーカーを落とす**
   - テスターで**無電圧確認（検電）**
2. **導通チェック**
   - 負荷側を**短絡（ショート）／ジャンパー**で1回路ループに
   - テスター**導通モード**（音 or 0Ω）で各端末を測定
   - 音が鳴らない → 不良で手直し
3. **端子・色・極性の照合**
   - 端子番号（A1/A2 等）・マーキング一致
   - 電線色（**黒=電源／白=N／赤=スイッチ**など）
   - 接地・極性が正しいか
4. **締付・固定**
   - 規定トルクで締付、**手で引っ張って緩みゼロ**
   - 端子台のネジ忘れがないか
5. **絶縁抵抗測定の準備**（次工程：021）
6. **記録**
   - **Before/After 写真**（特に制御線を外す時）
   - 結線図にチェック印
7. **他者確認**
   - 疲労時は必ず**他人の目**でテスターを当ててもらう

## まず見る図面・資料
1. **単線図（系統図）**：全体の流れ・回路番号 ← 最初の地図
2. **複線図・結線図・盤内詳細図（内部接続図）**：実際の端子接続・本数
3. 特記仕様書・設計変更指示書・質疑応答書（**最新変更**の反映）
4. 機器取扱説明書・メーカー図面（**一次側／二次側**・端子位置）
5. 総合図・平面図（位置・高さ・他業種干渉）
6. 自社の結線標準（電線色・マーキングルール）

## 誰に聞く？
- **元請け現場監督**（最終承認・連動試験スケジュール）
- 自社の先輩現場代理人・施工管理者
- 自社職長（社内の結線標準・トルク値）
- メーカー（特殊機器・制御盤の端子）
- 弱電・火報業者（連動回路の結線責任分担）

## 聞き方テンプレ
「**単線図と複線図**、両方ありますか？複線図がないと終端の本数が読めません」
「この回路の **無電圧確認**、立ち会ってもらえますか？」
「**端子番号と電線色の社内標準**を教えてください」
「**設計変更**は最新版で、この結線で合っていますか？」
「**規定トルク**は仕様書のどこですか？」
「疲れているので**他者目視確認**お願いします」

## 職長への指示テンプレ
「結線前に必ず**ブレーカー OFF＋無電圧確認**を」
「**負荷側ショート→テスター導通**で1回路ずつ確認してください」
「**端子番号・電線色・極性**を図面と照合、写真も残して」
「制御線を外す時は **Before 写真**を必ず、復旧用です」
「締付は **規定トルク＋引っ張りテスト**、緩みゼロで」
「**17時以降の作業**は他者確認を必ず入れてください」
「迷ったら**勝手に判断せず**連絡を」

## 事前チェック3つ
1. **単線図＋複線図＋取説**で結線の全体像と端子位置が把握できているか
2. **無電圧確認 → 導通 → 端子/色/極性 → 締付** の手順が身についているか
3. **写真（Before/After）と他者目視確認**で証拠と二重チェックが揃うか

> 補足：「**疲れた時の自分を信用しない勇気**」。17時以降のケアレスミスが年度末の敵。
> 補足：単線図だけでなく**複線図を頭で一筆書き**（電源→負荷→Nに戻る）でイメージする。
> 補足：**テスターの数値を信じる**。「大丈夫だろう」「勘」は最も危険。

## NG行動
- ブレーカーを落とさず結線する
- 検電（無電圧確認）を省略する
- テスター導通チェックをせず通電試験に入る
- 単線図だけで結線を完了したと判断する
- 制御線を写真なしで外す
- 端子番号・電線色を社内標準と照合しない
- 規定トルクを守らず手締めで済ませる
- 疲労時に一人で完結させる
- 設計変更を確認せず古い図面で進める

## 一言まとめ
結線確認は **「無電圧確認 → 負荷側ショート → テスター導通 → 端子/色/極性 → 締付 → 写真 → 他者目視」**の手順を崩さない。**疲れた自分を信用せず**、**単線図と複線図の両方**で全体を一筆書きで掴む。
