# 場面名
消防検査前に確認すべきことがわからない

## いつ起きる？
動作確認・連動試験完了後／消防検査の予定が組まれた時／消防設備士との現地リハーサル前／竣工検査の直前

## 消防検査とは（先に押さえる）
- **消防法に基づく検査**：自火報・非常放送・誘導灯・非常灯・消火栓・スプリンクラー・排煙・防火戸 等
- **書類検査がメイン＋現場確認**で構成
- 電気屋は **配線・結線・電源・連動** の責任 → ミスが**現場確認でバレやすい**
- **不合格＝再工事＋再検査**で、年度末は特にダメージ大
- 016（弱電火報の調整）と024（動作確認）の**結実点**

## 若手担当者が止まる理由
- **書類・試験結果**（絶縁・導通・動作）の揃え方が分からない
- 火報の **受信機・バッテリー容量・窓数・共通線**の用語と確認手順が頭に入っていない
- **電気屋の責任範囲 vs 消防設備士の範囲** が曖昧
- 消防署の**スケジュール（受付日・締切）**を把握していない
- 一人で完結しようとして消防設備士・監督への連絡が遅れる
- 5回線同時試験など**特殊な試験手順**を知らない
- 現地確認で何を見ればいいか具体化できない

## 怒られるポイント
- 配線・結線ミスが検査で発覚 → **「引き渡し前に最終確認しなかったのか」**
- 試験記録（絶縁抵抗値・動作試験）が**提出できない** → 書類検査で即NG
- 消防設備士・監督に連絡せず一人で「OK」判断 → 連動試験で失敗
- 現地確認を飛ばして書類だけ揃えた → 消防署到着後に**指摘されて信用失墜**
- バッテリー容量・共通線確認を省略 → 検査でやり直し
- 設計変更が書類に反映されていない → **古い書類で提出して大目玉**
- 「**ある日突然消防設備が使えなくなる**」原因が電気屋の結線間違い → **前回検査も不合格**だった事例

## 本当は確認すべきこと
1. **全前工程 100% 完了**
   - 結線・導通・絶縁・点灯・動作確認の **OK 記録**
   - 異常回路の手直し済み
2. **消防設備特有の最終チェック**
   - **火報（P型）**：受信機・バッテリー容量・**窓数**確認 → **全回路火災作動試験（5回線同時）** → **バッテリー稼働確認** → **共通線確認**
   - **非常放送**：簡易確認 → 現地放送試験
   - **消火栓・スプリンクラー**：起動試験、最後に**実報確認**
   - **非常灯・誘導灯**：常時点灯・停電点灯（バッテリー）
   - **連動**：火報 → 空調停止／シャッター／排煙／非常放送
3. **書類準備**
   - 試験結果（**絶縁抵抗値・導通記録・動作写真**）
   - 機器取説コピー
   - **設計変更反映済み**書類
   - 消防同意書・申請書類
4. **安全・連絡**
   - 周囲に「**消防検査準備中**」と周知
   - **消防設備士とペアで現地確認**（ベテラン＋若手パターンが鉄板）
5. **現地最終確認**
   - 消火栓ボックス内（**放水口・非常コンセント・ホース**）
   - **排煙起動スイッチ・防火戸**
   - 表示灯・地区音響の鳴動
6. **記録**
   - チェックリスト・写真・動画
   - 異常時は**即報告・手直し**

## 誰に聞く？
- **元請け現場監督**（検査スケジュール・全体調整）
- **消防設備業者・消防設備士**（火報・連動・書類の本丸）

## 聞き方テンプレ
「**消防検査のスケジュール**を教えてください、書類提出の締切も」
「**火報の5回線同時作動試験**、立会段取りお願いします」
「**バッテリー容量・窓数・共通線**の確認、消防設備士さんと一緒に見たいです」
「**書類**（絶縁・動作試験記録・設計変更反映）はどこに揃えますか？」
「電気屋の責任範囲と**消防設備士の範囲**、線引きを再確認したいです」
「**消火栓ボックス・排煙・防火戸**の現地リハーサル、いつやれますか？」

## 職長への指示テンプレ
「消防検査前に**結線・導通・絶縁・点灯・動作の OK 記録**を全部揃えてください」
「**火報受信機の窓数・バッテリー・共通線**を消防設備士と一緒に確認」
「**5回線同時の作動試験**を必ず実施、写真・動画を残す」
「**消火栓ボックス・排煙起動・防火戸**の現地確認を漏らさず」
「設計変更が**書類に反映**されているか、最終チェック」
「異常出たら**即連絡**、消防設備士さんに必ず共有」
「ベテランの消防設備士さんから**学ぶ姿勢**で同行してください」

## 事前チェック3つ
1. **全試験記録（020〜024 の OK 証拠）**＋**消防特有チェック（受信機・バッテリー・窓数・共通線・5回線・連動）**が揃っているか
2. **書類（絶縁値・動作写真・設計変更反映・取説）**が消防設備士・監督と確認できているか
3. **現地リハーサル**（消火栓・排煙・防火戸・連動先）の段取りと立会者が決まっているか

> 補足：**消防検査は書類がほぼ全て**＋現場確認は**確認程度**。ただし**電気屋の結線ミスはここでバレる**。
> 補足：**ベテラン＋若手ペア**が消防検査の鉄板。**学ぶ姿勢**で消防設備士に同行する。
> 補足：**「ある日突然消防設備が使えなくなる」原因のNo.1は電気屋の結線**。**最終確認**を侮らない。

## NG行動
- 試験記録を揃えずに検査日を迎える
- 消防設備士・監督に連絡せず一人で判断する
- 5回線同時作動試験・共通線確認を省略する
- 書類検査だけ済ませて現地確認を飛ばす
- 設計変更を書類に反映せず古い情報で提出する
- 消火栓ボックス・排煙・防火戸の現地確認を漏らす
- 消防署のスケジュール・締切を把握しない
- 異常を「様子見」で放置する

## 一言まとめ
消防検査は **「全試験記録 ＋ 受信機/バッテリー/窓数/共通線/5回線/連動 ＋ 書類（絶縁・動作・設計変更反映）＋ 現地リハーサル（消火栓・排煙・防火戸）」**。**書類が9割、現場確認で電気屋の結線ミスがバレる**。**消防設備士とペアで臨む**。
