# 場面名
施主検査前に準備することがわからない

## いつ起きる？
消防検査・動作確認完了後／引渡し直前のオーナー／施主立会検査の前日〜当日／是正対応の最終取りまとめ時

## 施主検査とは（先に押さえる）
- **オーナー／施主が立会で行う最終チェック**：機能・美観・使い勝手・位置までトータル
- **020〜025（試験〜消防検査）の集大成 ＋ 施主目線**
- 施主は **電気図面を持参して全部確認**してくることもある
- ここでの指摘＝**是正工事＋追加費用なし**で現場が背負う
- 美観や **「使い勝手」**が初めて評価される場 → 019（位置ズレ）の最終結審

## 若手担当者が止まる理由
- **施主が何を見るか**が予測できない（機能だけでなく美観・位置・操作感）
- **試験記録・書類**のまとめ方を経験していない
- 是正指摘が出た時の**直し順序・期限**のイメージが無い
- **設定（電気錠・インターホン・スマート設備）**が施主検査直前まで終わっていない
- 操作デモ（取説の読み込み）の準備が間に合わない
- 美観チェック（汚れ・ビス・プレート）の基準が曖昧
- 監督・施主への**立会連絡のタイミング**で迷う

## 怒られるポイント
- **コンセント位置が逆**（エアコン↔トイレ等） → 「点灯試験時に見逃したのか」
- スイッチ動作不良・照明のビス浮き・**プレート傾き** → 美観で大目玉
- **電気錠・インターホン・スマート機器の設定未了** → 施主が触って動かず**信用失墜**
- 試験結果・竣工図を**即提示できない** → 「準備不足」で是正命令
- 器具汚れ・ゴミ残し → 「引渡し前にこれか」
- 図面・打合せが曖昧なまま進めてきた結果 → **指摘が爆発**して**追加費用なしの直し**が降ってくる
- 操作説明できず施主の質問に答えられない → 「電気屋の責任」

## 本当は確認すべきこと
1. **全前工程 100% 完了**
   - 020〜025（結線・導通・絶縁・点灯・動作・消防）の **OK 記録**
   - 異常は**是正済み**
2. **機能の最終チェック（再度）**
   - 全照明・コンセント・スイッチの動作
   - **コンセント位置・高さ・個数**が図面通りか
   - 電気錠・インターホン・スマート設備・**非常灯の設定完了＆動作**
   - 調光・センサー・タイマーの動作
3. **美観チェック**
   - 器具清掃（拭き上げ・ホコリ除去）
   - **ランプ装着・プレート水平・ビス向き統一**
   - カバー・反射板の正位置、傷・汚れなし
4. **書類・記録準備**
   - **試験結果一式**（絶縁・導通・動作・点灯）
   - チェックリスト・是正リスト
   - **写真（Before/After）**
   - **取扱説明書・保証書**を施主用にまとめる
5. **施主目線リハーサル**
   - 部屋を一周、**施主のつもり**でスイッチを押す・座る・触る
   - **操作デモ**の練習（取説読込）
6. **連絡・段取り**
   - 監督に「**施主検査準備完了、立会お願いします**」
   - 是正指摘が出た時の**直し順序・期限**を事前に合意

## まず見る図面・資料
1. **電気図面一式**（単線図・分電盤図・**照明配置図・竣工図／as-built**）← まずこれ。施主も持参するケースあり
2. **機器取扱説明書・操作説明書・保証書**（操作デモ・施主向け説明）
3. 特記仕様書・設計変更指示書・**試験記録（絶縁／導通／動作／点灯）**
4. **施主検査用チェックリスト・是正リスト**
5. 総合図・防火設備図（連動の最終突き合わせ）
6. **019 で整理した位置ズレ防止チェック**

## 誰に聞く？
- **元請け現場監督**（立会・是正対応の窓口）
- 自社の先輩現場代理人・施工管理者
- 自社職長（清掃・美観の社内ルール）
- メーカー（電気錠・インターホン・スマート設備の設定）
- 弱電・火報業者（連動先の最終確認）

## 事前チェック3つ
1. **020〜025 の OK 記録 ＋ 機能・美観・設定の最終リハーサル**が完了しているか
2. **電気図面一式・取扱説明書・保証書・試験記録**が施主提示用にまとまっているか
3. **是正指摘の受け方（記録・期限・段取り）**が監督と合意できているか

> 補足：施主は **電気図面を持参して全部確認**してくることがある。**こちらも完璧な一式**で迎える。
> 補足：機能だけでなく **「美観・使い勝手」** が初めて評価される。**プレート水平・ビス向き・清掃**を侮らない。
> 補足：是正指摘＝**追加費用なしで自分達が背負う**。**事前リハーサルで指摘を減らす**のが最大の節約。

## NG行動
- 020〜025 の OK 記録を揃えずに当日を迎える
- 機能チェックだけで美観・清掃を省略する
- 電気錠・インターホン・スマート機器の設定を当日まで放置する
- 取扱説明書・保証書を施主用にまとめない
- コンセント位置・高さの最終確認を省略する
- 操作デモの練習をせず施主の質問に答えられない
- 監督への立会連絡を忘れる
- 指摘を口頭で済ませて記録に残さない
