# 場面名
点灯試験で事前に確認すべきことがわからない

## いつ起きる？
結線・導通・絶縁測定が完了した直後／受電後の初投入時／竣工検査前の最終確認時／非常灯・調光・センサー器具の動作確認時

## 点灯試験とは（先に押さえる）
- 照明回路に**実際に通電して器具を点灯させる**最終工程
- **020〜022（結線・導通・絶縁）の集大成**
- 「**パチっと明かりが灯った瞬間に現場の空気が『完成』に変わる**」プロの達成感の場
- 反面、ミスると**ショート・トリップ・感電・器具焼損・再工事**で一気に信用失墜

## 若手担当者が止まる理由
- 前工程が**本当に全部終わっているか**自信が持てずブレーカーに触れない
- **投入順序**（全回路一括 / 系統ごと / 1回路ずつ）の判断ができない
- **非常灯・制御回路・調光器**など特殊回路の手順が分からない
- ランプ未装着・ワット数違い・バッテリー未接続など**負荷側の最終確認**を見落とす
- **周囲への周知**（「これから点灯します」）が言い出せない
- 異常（チカチカ・異音）が出た時の**判断・手順**を持っていない
- 一人でやるべきか・先輩立会か迷う

## 怒られるポイント
- 前工程飛ばしで送電 → **ショート／トリップで再工事**
- ランプ未装着・**非常灯バッテリー未接続** → 「半分しか点かない」
- **仮設ジャンパー除去忘れ** → 想定外の挙動
- 周知なしで突然投入 → 他業種に迷惑、感電リスク
- 異常を「**様子見**」で放置 → 焼損・火災に発展
- 顧客目線確認を省略 → ビス向き・汚れ・スイッチ感で施主クレーム
- 記録・写真なしで「点いた」と報告 → 後でどこか分からない
- **非常灯の電圧チェック忘れ** → **感電事故**

## 本当は確認すべきこと（事前確認手順）
1. **全前工程完了の確認**
   - **結線・導通・絶縁抵抗測定が100% OK**
   - 異常があった回路は**手直し済み**
2. **器具・負荷側の最終チェック**
   - **ランプ装着**・ワット数・種類適合
   - 器具内の**清掃・異物なし**
   - **カバー・反射板**正位置
   - **非常灯はバッテリー接続済み**
3. **配線・スイッチ確認**
   - **仮設ジャンパー全除去**
   - 線番・色一致、極性
   - スイッチ・**調光器・センサー**の動作準備
4. **安全確保**
   - 対象回路の**ブレーカー全OFF → タグアウト**
   - **周囲に大声で周知**「これから点灯試験します」
   - テスターで**最終無電圧確認**
5. **試験実施**
   - **1回路ずつ（または系統ごと）**にブレーカー投入
   - 点灯確認＋**異常（異音・異臭・発熱・チカチカ）**チェック
   - 部屋を一周して**顧客目線**（ビス向き・汚れ・スイッチ押し心地）
6. **記録**
   - **点灯写真・回路番号・日時・作業者**
   - 異常時は**即停止・報告・手直し**

## まず見る図面・資料
1. **照明器具配置図・平面図** ← まずこれ（位置・種類・数量）
2. **単線図（系統図）・分電盤図**（回路番号・ブレーカー対応・非常灯回路）
3. **器具表・機器取扱説明書・メーカー図面**（ランプ仕様・非常灯バッテリー）
4. 特記仕様書・設計変更指示書・質疑応答書（**調光制御・特殊器具**）
5. 総合図・**スイッチングスケジュール**（操作系統）
6. 020〜022 の試験記録（**前工程の OK 証拠**）

## 誰に聞く？
- **元請け現場監督**（投入承認・スケジュール・立会）
- 自社の先輩現場代理人・施工管理者（**立会必須**）
- 自社職長（投入順序・社内ルール）
- メーカー（**特殊器具・非常灯・調光**の手順）
- 弱電・火報業者（連動先の挙動確認）

## 職長への指示テンプレ
「点灯試験前に**結線・導通・絶縁の OK 記録**を全部揃えてください」
「**ランプ装着・ワット数・カバー・非常灯バッテリー**を全器具で再確認」
「**仮設ジャンパー除去**を必ず最終確認」
「投入は**1回路ずつ**、**異音・異臭・発熱・チカチカ**を見逃さない」
「**周囲に大声で周知**してから投入してください」
「異常出たら**即停止・連絡**、様子見は禁止」
「点灯後は**部屋を一周、顧客目線**で最終チェック」

## 事前チェック3つ
1. **020〜022（結線・導通・絶縁）の試験記録**が全回路 OK で揃っているか
2. **ランプ・カバー・非常灯バッテリー・仮設ジャンパー除去**など負荷側の最終確認が完了したか
3. **周囲周知＋立会＋投入順序＋異常時の戻し手順**が合意できているか

> 補足：点灯試験は**全前工程の集大成**。前工程に1つでも未確認があると、すべてが崩れる。
> 補足：**疲労時は絶対一人でやらない**。先輩 or 監督立会で。
> 補足：**非常灯の電圧チェック忘れは感電事故**につながる。バッテリー接続を侮らない。

## NG行動
- 結線・導通・絶縁の OK 記録なしで投入する
- ランプ未装着・カバー未取付のまま試験する
- 仮設ジャンパーを除去せず投入する
- 周囲に周知せず突然投入する
- 異常（チカチカ・異音）を「様子見」で放置する
- 一人で全部やってしまう
- 顧客目線の最終チェックを省略する
- 写真・回路番号・日時を記録しない
- 非常灯のバッテリー・電圧確認を省略する
