# 場面名
是正対応で優先順位の付け方がわからない

## いつ起きる？
施主検査・消防検査・社内検査の指摘が複数出た直後／是正期間中の毎朝の段取り時／引渡し直前のラストスパート時

## 是正対応とは（先に押さえる）
- 検査で出た**指摘の不具合を直す**作業フェーズ
- 出元：**施主検査・消防検査・社内検査・元請け検査**
- **追加費用なし＋短い期限**で現場が背負う
- 優先順位を間違えると **工期遅れ・再検査NG・信用失墜**
- **025〜026 の結果を、引渡しに繋げる最終工程**

## 若手担当者が止まる理由
- 指摘が **10件以上**出た時、どれから手を付ければいいか分からない
- **「全部同時進行？」**で混乱して動けない
- 「電気的には問題ないのにやり直し」指摘に**納得できず手が動かない**
- 工期・人員不足のプレッシャーで一人で抱え込む
- 是正前後の**記録（写真・測定値）の段取り**ができない
- **完了報告**のタイミング・方法が分からない
- 上司に「**優先順位これでいいですか？**」と聞けない

## 怒られるポイント
- **美観から先に手をつけて安全・機能を後回し** → 「**優先順位おかしいだろ！**」
- 是正前後の写真がない → 「**直した証拠がない**」と信用ゼロ
- 「電気的にOK」と渋って渋々やり直し → **態度**で怒られる
- 是正期間カツカツで**夜間・残業連発** → 職人疲労でミス連発
- 監督・上司への進捗報告なし → 「**何やってるか分からない**」
- 完了報告書なし → 引渡しに進めない
- 是正中に新たなミスを生む → **二重の信用失墜**

## 本当は確認すべきこと（優先順位の鉄則）
1. **最優先 A：安全・法令関連**
   - 感電・絶縁不良・接地不良
   - **火報連動不良・耐火電線未使用**
   - 消防検査 NG 項目（再検査回避）
   - **直さず進むと事故・法令違反**
2. **優先 B：機能・動作不良**
   - 点灯しない／スイッチ逆転／**コンセント位置逆**
   - 調光・センサー・連動が動かない
   - 施主が**毎日使う部分**
3. **優先 C：納まり・仕様違反**
   - 図面と違う高さ・位置
   - 締付緩み・支持不足
   - 仕様書の規格未達
4. **最後 D：美観・軽微**
   - **プレート傾き・ビス浮き・汚れ**
   - 時間がある時に対応

## 是正対応の標準手順
1. **指摘リスト分類**：A/B/C/D で優先度付け（Excel・チェックシート）
2. **段取り**：監督・上司に「**優先順位これで進めます**」と確認
3. **記録**：**是正前写真 → 作業 → 是正後写真＋測定値**
4. **進捗報告**：毎朝／毎日、進捗を共有
5. **完了報告**：「**是正完了報告書**」提出
6. **疲労時**：**絶対一人でやらずペア作業**

## まず見る図面・資料
1. **是正指摘リスト・検査指摘書**（施主／消防／社内）← まずこれ
2. 電気図面一式（**as-built 竣工図・単線図・照明配置図**）
3. 特記仕様書・設計変更指示書・**消防同意書**（法令・仕様基準）
4. **前工程記録**（絶縁・導通・動作試験結果）：是正前後の測定値比較
5. **是正完了報告テンプレート**（自社・提出用）

## 誰に聞く？
- **元請け現場監督**（優先順位の最終承認・期限合意）
- 自社の先輩現場代理人・施工管理者
- 自社職長（人工配置・夜間段取り）
- メーカー（特殊機器の是正方法）
- 消防設備業者（消防指摘の是正範囲）
- 設計担当（仕様解釈の不明点）

## 聞き方テンプレ
「指摘リストを **A/B/C/D で分類**しました、**優先順位これでいいですか？**」
「**是正期限**はいつまでですか？人工と工程を相談したいです」
「**安全関連の A 項目**を最優先で進めます、立会お願いできますか？」
「『電気的には問題ない』と思う指摘がありますが、**仕様の意図**を教えてください」
「**是正前後の写真フォーマット**、社内ルールありますか？」
「**完了報告書**のテンプレートをください」

## 職長への指示テンプレ
「指摘は **A（安全）→ B（機能）→ C（納まり）→ D（美観）**の順で進めてください」
「**A 項目（安全・法令）は今日中**、B は明日中、C/D は今週中」
「**是正前後の写真**を必ず残してください、証拠が無いと『直した』と言えません」
「**夜間・疲労時は一人作業 NG**、ペアで」
「『電気的にOK』と思っても、**仕様書通りに直す**のが鉄則」
「進捗を**毎朝報告**、新たなミスを生まないように」

## 事前チェック3つ
1. **指摘リスト**を A（安全）/ B（機能）/ C（納まり）/ D（美観）で分類し、監督と合意できているか
2. **是正前後の写真・測定値の記録**段取りができているか
3. **期限・人工配置・夜間ペア作業**の段取りが組めているか

> 補足：**「安全 ＞ 機能 ＞ 納まり ＞ 美観」**の優先順位は**絶対の鉄則**。順序が逆になると全部やり直し。
> 補足：「電気的には問題ない」と感じても、**仕様書通りに直す**。納得できなくても**手は動かす、抗議は記録に残す**。
> 補足：**是正中に新たなミスを生むのが最悪**。**ペア作業＋進捗報告**で二重事故を防ぐ。

## NG行動
- 美観から先に手を付ける
- 「電気的にOK」と渋って手を動かさない
- 是正前後の写真を撮らない
- 進捗報告を省略する
- 完了報告書を出さない
- 一人で抱え込んで夜間連発
- 優先順位を上司に確認せず勝手に進める
- 是正中の新たなミスを隠す

## 一言まとめ
是正対応は **「A：安全 → B：機能 → C：納まり → D：美観」**の優先順位を**絶対**に守る。**指摘リスト分類 → 監督合意 → 是正前後の写真 → 毎日進捗 → 完了報告**の段取りで、**疲労時はペア作業**。
