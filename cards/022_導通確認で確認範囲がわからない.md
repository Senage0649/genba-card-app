# 場面名
導通確認で確認範囲がわからない

## いつ起きる？
結線完了後の自主検査時／絶縁測定の前後／通電試験の直前／3路スイッチや制御回路の動作前確認時／改修・リフォーム現場の既設線調査時

## 導通確認とは（先に押さえる）
- テスター（DMM）の**導通モード**（音 or 0Ω）で**配線の「つながり」と「正しさ」**を確認する作業
- 必ず**無電圧状態**で実施（活線で触らない）
- **断線・誤結線・短絡**を通電前に発見する最後の砦
- 020（結線確認）・021（絶縁測定）と**3点セット**で扱う

## 若手担当者が止まる理由
- 「**どこからどこまで測ればいいか**」範囲がイメージできない
- **3路スイッチの連絡線（トラベラー）**でどっちがどっちか迷う
- 旧配線・改修現場で**色がバラバラ**で識別できない
- **テスターの電池切れ**（予備電池忘れ）で工程ストップ
- 図面と現場のズレで「これで合ってる？」と止まる
- **負荷側を短絡（ジャンパー）するか一本ずつか**で判断停止
- **異相間・対地間の意図しない導通（短絡）**を確認する発想がない

## 怒られるポイント
- 通電前に導通確認をせず「まぁ大丈夫」 → **後工程で断線発覚 → 再工事**
- **変更後配線図を貰っていない**まま勘で作業 → 「**配線図通りにやれ**」
- 活線で触る／無電圧確認なし → **火災・感電リスク**で信用失墜
- 異常を記録せず・写真もなし → **どこが悪いか調査が長引く**
- 範囲漏れ（消防・制御回路を測り忘れ） → **連動試験で不良発覚**
- LANケーブル等で**半挿し Pass**を見抜けない → 仕上げ後に断続不良
- 疲労時の電池切れで工程停止 → 「準備不足」と叱責

## 本当は確認すべきこと（確認範囲）
1. **安全確保（最優先）**
   - 回路電源OFF → **タグアウト／ロックアウト**
   - テスターで**無電圧確認（検電）**、周囲連絡
2. **負荷側短絡（ジャンパー）**
   - 負荷側（照明・コンセント等）を一時的にくっつけて**1回路ループ**に
3. **テスター導通チェック（範囲）**
   - **同一導体**：電源 → 負荷 → N の**一筆書き**で断線なし
   - **各回路ごと**：分電盤主幹 → 分岐 → 機器終端まで**全部**
   - **特殊回路**：3路スイッチの**連絡線（トラベラー）**、制御線、リレーコイル、**消防の遮断弁回路**
   - **異相間・対地間**：**意図しない導通（短絡）が無い**ことも確認
4. **付随確認**
   - **線番・色**が図面と一致
   - 締付・端子の差し込み深さ（**半挿し Pass を疑う**）
5. **記録**
   - **Before/After 写真**、測定値・回路名メモ
   - 異常時は**即報告・手直し**、隠さない
6. **他者確認**
   - 新人時は先輩に「**この範囲でカバーしてる？**」と見てもらう

## まず見る図面・資料
1. **単線図（系統図）**：回路番号・全体の流れ ← まずこれ
2. **複線図・結線図・盤内詳細図・分電盤図**（端子位置・導通範囲・線番）
3. **特記仕様書・設計変更指示書・質疑応答書**（最新変更）
4. 機器取説・メーカー図面（**3路スイッチ・制御機器の端子仕様**）
5. 総合図・平面図（他業種干渉・終端位置／**特にリフォーム現場**）
6. 自社の導通確認チェックリスト

## 誰に聞く？
- **元請け現場監督**（最終承認・立会・連動試験スケジュール）
- 自社の先輩現場代理人・施工管理者
- 自社職長（測定範囲・社内チェックリスト）
- 弱電・火報業者（連動回路の確認範囲分担）
- メーカー（特殊機器の端子仕様）

## 聞き方テンプレ
「**変更後の配線図**は最新版どこにありますか？」
「3路スイッチの**連絡線（トラベラー）**、社内ではどう識別していますか？」
「**消防・制御回路の導通確認範囲**は誰がどこまでやりますか？」
「**負荷側ジャンパー**の社内ルール（一括 / 一本ずつ）を教えてください」
「**テスターの予備電池**、現場に共通のがありますか？」
「**他者目視確認**お願いできますか、疲れているので」

## 職長への指示テンプレ
「導通確認前に**電源OFF＋タグアウト＋無電圧確認**を必ず」
「**1回路ずつ**、電源 → 負荷 → N の一筆書きで断線確認」
「**3路スイッチ・制御線・消防回路**は範囲漏れに注意」
「**異相間・対地間**の意図しない導通（短絡）も必ず測ってください」
「**半挿し Pass** に注意、しっかり差し込んでから測る」
「**予備の9V電池**を腰袋に必ず忍ばせて」
「異常出たら**即連絡、隠さず報告**」

## 事前チェック3つ
1. **電源OFF＋タグアウト＋無電圧確認**の安全手順が揃っているか
2. **同一導体／各回路／特殊回路（3路・制御・消防）／異相・対地短絡** の4範囲を網羅できているか
3. **テスター電池（予備含む）・線番識別・写真記録・他者確認**の段取りができているか

> 補足：**「配線の『つながり』と『正しさ』を100%証明する」**のが導通確認の目的。
> 補足：**3路スイッチの連絡線**（トラベラー）は迷ったら**両端短絡＋テスターで識別**。
> 補足：疲労時は**予備9V電池**＋**他者確認**。一人で完結させない。

## NG行動
- 無電圧確認をせず触る
- 通電前に導通確認を省略する
- 同一導体の一筆書きしか測らず、異相・対地短絡を見落とす
- 3路スイッチ・制御・消防回路を範囲外にする
- 半挿しのまま「導通あり」と判定する
- 変更図面を確認せず古い情報で作業する
- 異常を記録せず・写真も残さない
- 電池切れで止まる（予備なし）
- 異常を隠蔽する

## 一言まとめ
導通確認は **「無電圧確認 → 負荷側ジャンパー → 同一導体／各回路／特殊回路（3路・制御・消防）／異相・対地短絡 の4範囲を網羅 → 写真・他者確認」**。**配線の『つながりと正しさ』を100%証明**する最後の砦。
