# 場面名
動作確認で誰を呼べばいいかわからない

## いつ起きる？
点灯試験完了後／制御盤・ポンプ・空調機等の試運転時／消防設備・自火報の連動試験時／竣工検査前の最終動作確認時

## 動作確認とは（先に押さえる）
- 機器・回路を**実際に動かして機能・連動が仕様通りか**確認する工程
- **「全工程の最終証明」**：023（点灯試験）の次のステップ
- 単独動作 → 連動無負荷 → 連動負荷 → 引渡しの順で進む（試運転シーケンス）
- **絶対に一人で完結させない。関係者を呼ぶのが鉄則**

## 若手担当者が止まる理由
- **照明 / 消防 / 機械設備 / 制御盤**で呼ぶ相手が違うのに、判断軸がない
- **連動動作の範囲・タイミング**が読めず、誰がいつ来るか把握できない
- **試運転の音頭を取る**のが電気屋の仕事、と知らずに待機してしまう
- 前工程（導通・絶縁・点灯）が本当に終わったか自信が持てない
- **立会の依頼の仕方**（時間・連絡方法）が分からない
- 「自分のせいで他業者を呼んでいいのか」と気後れする

## 怒られるポイント
- **一人で勝手に動作確認** → 連動ミスで再工事、「**なぜ連絡しなかった**」
- 施主・メーカー不在で「正常動作した」と報告 → **証拠不足で後日再確認命令**
- 記録・写真・動画なし → 「**証拠がない**」と叱責
- 電気工事が終わっていないのに試運転 → 他業者に迷惑
- 異常を「様子見」で放置 → 焼損・事故リスク
- **連動表を見ずに**部分確認だけ → 連動先で動作不良
- 機械屋・メーカーを呼ぶタイミングを逃す → 工程後ろ倒し

## 誰を呼ぶか（優先順位マップ）
**Step 1：自分側を固める**
- **自社の先輩・電気工事責任者（電気主任技術者）**：若手はここから。「今から動作確認、立会お願いします」
- **自社職長**（試運転手順の社内ルール）

**Step 2：元請けに通す**
- **元請け現場監督**（必須）：全体調整・スケジュール承認

**Step 3：機能・回路ごとの専門家**
- **照明・コンセント・スイッチ**：監督立会で OK
- **消防・自火報**：**消防設備業者／消防設備士／消防署**（消防法関連）
- **制御盤・ポンプ・空調機**：**メーカー＋機械設備業者**
- **連動（火報→空調停止／シャッター／非常放送）**：弱電・火報・空調・防災の各業者
- **特殊機器・調光・センサー**：メーカー（試運転マニュアル）

**Step 4：引渡し関係**
- **施主・管理会社・設計者**：引渡し前の最終立会

**音頭は電気屋が取る**：電気屋が前工程 OK を出して初めて他業者が動ける。**待つのではなく呼ぶ**。

## 本当は確認すべきこと
1. **前工程 100% 完了**
   - 020〜023（結線・導通・絶縁・点灯）の OK 記録
   - 異常回路の手直し済み
2. **チェックリスト準備**
   - 正常動作・異常動作・**連動確認**・異常音／発熱／振動
3. **試運転シーケンス**
   - **単独動作 → 無負荷試運転 → 連動無負荷 → 連動負荷 → 引渡し**
4. **立会者の合意**
   - 誰が・いつ・何を確認するか **連動表で割付**
5. **記録**
   - **写真・動画・測定値・回路番号・日時・立会者**
6. **異常時の戻し手順**
   - どこまで戻して手直しするかを事前に合意

## まず見る図面・資料
1. **動作仕様書・連動表・シーケンス図** ← まずこれ（**誰が何を確認するか**の地図）
2. 単線図・分電盤図・**制御盤詳細図**（回路・端子）
3. **機器取説・メーカー図面・試運転マニュアル**（正常動作条件）
4. 特記仕様書・設計変更指示書・**竣工検査要領**（立会ルール）
5. 総合図・他業種調整図（連動設備の確認範囲）
6. 020〜023 の試験記録（**前工程 OK 証拠**）

## 誰に聞く？
（誰を呼ぶかマップを参照）
- **元請け現場監督**（最優先・全体調整）
- 自社の先輩・電気工事責任者
- 消防設備業者・消防設備士
- 機械設備業者・メーカー
- 弱電・火報・空調・防災業者
- 施主・管理会社・設計者

## 聞き方テンプレ
「動作確認の**立会スケジュール**を組みたいです、いつが空いていますか？」
「**連動表**で立会が必要な業者を割付したいです、確認お願いします」
「**消防の連動試験**は消防設備業者を呼べばいいですか？消防署立会も必要ですか？」
「**制御盤の試運転**、メーカー立会の手配お願いできますか？」
「電気側は前工程 OK です。**機械屋さんを呼ぶ音頭**を取りたいです」
「異常出たら**どこまで戻して**手直しするか、事前に決めておきたいです」

## 職長への指示テンプレ
「動作確認は**前工程 OK 記録**を全部揃えてから」
「**単独 → 無負荷 → 連動無負荷 → 連動負荷**の順で進めてください」
「**写真・動画・測定値**を必ず残してください」
「**立会者**（監督・先輩・メーカー・施主）の到着を確認してから開始」
「異常出たら**即停止・連絡**、様子見禁止」
「**連動先（空調停止・シャッター・非常放送）**まで全部見届けて」
「**疲労時・夜間は絶対一人でやらない**」

## 事前チェック3つ
1. **連動表＋試運転マニュアル＋前工程 OK 記録**が揃っているか
2. **誰が・いつ・何を確認するか**を立会者全員で合意できているか
3. **異常時の戻し手順＋写真・動画・測定値の記録**の段取りが組めているか

> 補足：**音頭は電気屋が取る**。電気が OK を出して初めて他業者が動ける。**待たずに呼ぶ**。
> 補足：**動作確認＝全工程の最終証明**。証拠（写真・動画・測定値）が無いと「やった」と言えない。
> 補足：消防・制御・特殊機器は**専門家立会必須**。一人で「動いた」と判断しない。

## NG行動
- 一人で動作確認を完結させる
- 連動表を見ずに単独動作だけで OK と判断する
- 立会者を呼ぶタイミングを逃す
- 写真・動画・測定値を残さない
- 異常を「様子見」で放置する
- 前工程の OK 記録を確認せずに開始する
- 試運転シーケンスを飛ばす（いきなり連動負荷）
- 引渡し前の施主立会を省略する

## 一言まとめ
動作確認は **「連動表で立会者を割付 → 単独 → 無負荷 → 連動無負荷 → 連動負荷 → 引渡し」**。**音頭は電気屋が取る**。**証拠（写真・動画・測定値）と立会**で「やった」を証明する。
