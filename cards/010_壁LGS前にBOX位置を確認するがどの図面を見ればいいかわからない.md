# 場面名
壁LGS前にBOX位置を確認するがどの図面を見ればいいかわからない

## いつ起きる？
LGS（軽量鉄骨）立て込み前の墨出し・位置確認時／LGS業者との事前調整時／壁内配線・BOX建て込み準備時／ボード張り直前の最終確認時

## LGS と BOX とは（先に押さえる）
- **LGS（軽量鉄骨下地）**：壁の骨組み。**ランナー（上下）＋スタッド（縦）**で構成
- **BOX**：コンセント・スイッチ・通信などの**埋込BOX**（後からプレートを付ける箱）
- LGS前は **「位置墨出し・LGS業者への事前調整・プロット確認」** が中心
- LGS後（ボード張り前）が **BOX建て込みの本番**

## 若手担当者が止まる理由
- 「**どの図面に BOX 位置が書いてあるか**」を知らない（電気設備図だけ見て止まる）
- **高さ基準（FL+〇mm）**が頭に入っていない
- **LGSスタッドと BOX が干渉する**という発想がない
- 他職種（衛生・設備）の水栓・配管との**取合い**まで意識できない
- 「図面通りやれば正解」と思い込み、現場条件を確認しない
- 確認したいのに「展開図にない」と止まり、誰に聞けばいいか分からない

## 怒られるポイント
- **承認図を見ずに先に BOX を入れる** → 水栓・配管位置と被って衛生屋から大揉め
- 図面と逆位置に取り付け → **施主から「一生不便になる」と大目玉**
- 高さ基準（FL+〇mm）を確認せず「いつもの位置」 → 「**仕様書読んだのか**」
- LGSスタッドと干渉 → BOXが取り付けられず、LGS業者にやり直し依頼
- ボード張り後に位置ズレ発覚 → クロス屋・大工から総叩き
- 「赤マッキーで印つけたのに壁仕上がってる」 → 確認漏れの責任追及
- 黙って止まる → 「なぜなぜ攻撃」される前に**先に聞きに行け**

## 本当は確認すべきこと
1. **平面位置**：通り芯・壁中心からの寸法
2. **高さ**：FL（床仕上げ面）からの基準寸法（例：コンセント FL+300mm、スイッチ FL+1100mm）
3. **向き・間隔**：BOX の連装方向、複数 BOX の中心間距離
4. **LGSスタッドとの干渉**：スタッドピッチ（一般 303 or 455mm）との位置関係
5. **他職種との取合い**：給排水・水栓・空調吹出口との競合
6. **図面の最新版**：改訂・施主変更の反映確認
7. **現場墨出しとの一致**：地墨・レベル墨と図面寸法の整合

## まず見る図面・資料（優先順）
1. **電気設備図（機器配置図・器具配置図）** ← まずこれ：BOX の平面位置
2. **壁展開図（意匠展開図）**：壁面での**高さ・横位置の詳細**
3. **特記仕様書・内線規程**：**高さ基準**（FL+〇mm の指定）
4. 建築意匠図（平面・立面）：部屋レイアウト・家具・ドア位置
5. 現場の **墨出し資料・プロット図**：図面と現場のズレを直接確認
6. 施工図（最新承認版）：版次・施主変更の反映

## 誰に聞く？
- **元請け現場監督**（最終承認役・他職種調整）
- 自社の先輩現場代理人・施工管理者
- **LGS業者の職長**（スタッドピッチ・干渉確認）
- 衛生・空調職長（水栓・吹出口との取合い）
- 自社職長（プロット確認・墨出し）

## 聞き方テンプレ
「BOX 位置の確認で、電気設備図を見ています。**展開図と特記仕様書も併せて確認**した方がいいですか？」
「**コンセント・スイッチの高さ基準**は仕様書のどこに書いてありますか？（FL+〇mm）」
「LGSのスタッドピッチを教えてください。BOXと干渉しないか確認したいです」
「水栓・配管位置とBOXがぶつかりそうです。先に取合いを調整させてください」
「この施主変更は最新図面に反映されていますか？」

## 職長への指示テンプレ
「BOX は **電気設備図＋展開図＋特記仕様書（FL+〇mm）** で位置・高さを決めてください」
「**LGSスタッドと干渉する箇所**は、勝手に動かさず必ず連絡してください」
「衛生の水栓・配管と近接する箇所は、先に取合い打合せをしてから建て込んでください」
「LGS業者と**事前に位置を共有**してから墨を出してください」
「ボード張り前に**位置・高さ・向きの写真**を残してください」

## 事前チェック3つ
1. **電気設備図＋壁展開図＋特記仕様書**の3点を揃え、平面位置・高さ・向きが明確になっているか
2. **LGSスタッドピッチ**と**他職種（衛生・空調）の取合い**を事前に調整したか
3. 図面が**最新版**（施主変更反映済み）で、現場墨出しと一致しているか

> 補足：高さ基準の一般例（仕様書で要確認）：**コンセント FL+300mm／スイッチ FL+1100mm**。
> 補足：LGSスタッドピッチは一般 **303mm or 455mm**。BOX が被ったらLGSと事前調整。
> 補足：「**展開図にない**」時は黙らず、監督に「これどこ見れば？」と先に聞く。

## NG行動
- 電気設備図だけで判断する（展開図・仕様書を見ない）
- 高さ基準を「いつもの位置」で済ませる
- LGSスタッドとの干渉を確認しない
- 他職種との取合い打合せを省略する
- 承認図を確認せず先にBOXを入れる
- 施主変更が反映された最新版か確認しない
- 黙って止まる（聞きに行かない）
- 赤マッキーの印・指示メモだけで現物確認しない

## 一言まとめ
BOX位置は **電気設備図＋壁展開図＋特記仕様書（高さ基準）** の3点セットで決め、**LGSスタッド・他職種との干渉**を事前に潰す。**LGS前は事前調整、LGS後はボード張り前が本番**。
