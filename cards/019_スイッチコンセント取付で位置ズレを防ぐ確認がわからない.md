# 場面名
スイッチコンセント取付で位置ズレを防ぐ確認がわからない

## いつ起きる？
BOX取付後・壁ボード張り後の器具取付直前／墨出し・レベル確認時／施主変更（家具・エアコン位置）の反映時／引渡し前の最終確認時

## スイッチコンセント取付とは（先に押さえる）
- BOX に **スイッチ・コンセント・通信プレート**を取り付け、壁面で**毎日使う最終仕上げ**
- 「**毎日使う顔**」：施主・住人が**ズレを一番感じる**部位
- 1mm のズレが積み重なると**水平・連用ガタつき**で見栄え激落ち
- やり直しは**壁穴・ボード補修**まで波及して**手戻り地獄**

## 若手担当者が止まる理由
- 図面位置と現場墨が合わず、どちらを優先すべきか分からない
- 高さ基準を「**いつもの位置**」で済ませる
- 壁面からの**離れ寸法**（ドア枠・コーナーから何mmか）が頭にない
- BOX 位置・向きと取付器具の連動を確認しない
- 家具・エアコン・棚レイアウトとの**干渉まで想像**できない
- **施主変更**の反映漏れに気づかない
- 連用枠・ON/OFF 方向・接地極位置などの細かい仕様で止まる

## 怒られるポイント
- **位置逆・高さズレで施主大クレーム** → やり直し＋ボード補修
- スイッチが**利き手逆側／ドア枠に近すぎて使いづらい** → 「**電気屋の仕事だろ**」
- コンセントが**家具裏に埋もれる** → 日常ストレスでクレーム
- BOX とのズレで**配線届かず干渉** → 取付不能で詰む
- 連用枠に**隙間・ガタつき** → 見栄えで施主から指摘
- **エアコン位置変更**を反映できず → ハウスメーカーから怒られる
- 水平・垂直のレベルが取れていない → 一目で「下手」と判定

## 本当は確認すべきこと
1. **位置・高さ・左右離れ**
   - 平面位置（BOX 中心）
   - 高さ：仕様書通り（一般 **コンセント FL+250〜300mm／スイッチ FL+1,100〜1,200mm**）
   - 壁面からの**水平離れ**（ドア枠・コーナーから **100mm 以上**目安）
2. **BOX 連動・レベル確認**
   - BOX 位置・向きと一致しているか
   - **水平器**で垂直・水平を確認
3. **他職種・現場変更反映**
   - 家具・エアコン・棚レイアウトとの干渉
   - **施主変更（エアコン位置・棚位置）が最新図面に反映**されているか
   - 現場墨出し・レベル墨と図面の照合
4. **向き・連用枠**
   - スイッチ ON/OFF 方向
   - **コンセント接地極（アース側）の位置**
   - 複数連用時の**隙間ゼロ**
5. **最終作動・安全**
   - **検電後の通電確認**
   - 固定強度（緩みゼロ）
   - 連用枠のビス留めトルク

## まず見る図面・資料
1. **壁展開図（意匠展開図）＋電気設備図（スイッチ・コンセント配置）** ← まずこれ
2. 機器配置図・特記仕様書（**高さ基準・離れ基準・連用ルール**）
3. 建築平面図・立面詳細図（ドア・家具・他設備との干渉）
4. 現場墨出し資料・**レベル墨・改訂図面**
5. 器具取扱説明書・内線規程（取付方法・向き）
6. 施主変更履歴（**エアコン位置・家具配置の最終版**）

## 誰に聞く？
- **元請け現場監督**（最終承認・施主変更の反映）
- 自社の先輩現場代理人・施工管理者
- 自社職長（高さ・離れの社内ルール）
- 内装・家具担当（家具との干渉確認）
- 空調業者（**エアコン位置の最終確定**）
- 施主・設計担当（変更点の反映確認）

## 聞き方テンプレ
「コンセント・スイッチの**高さ基準**は仕様書のどこですか？（FL+〇〇mm）」
「**ドア枠・コーナーからの離れ**は何mmですか？」
「**エアコン位置の最終確定**はこの図面で合っていますか？」
「家具・棚との干渉、施主変更の**最新版**を確認したいです」
「壁貼り後の**位置確認**、立ち会ってもらえますか？」
「連用枠の**社内標準ビスとトルク**を教えてください」

## 職長への指示テンプレ
「取付前に**展開図＋仕様書＋現場墨**で位置・高さを必ず照合してください」
「**水平器**で水平・垂直を必ず取って、ガタつきゼロで」
「**ドア枠・コーナーから 100mm 以上**離してください」
「**コンセント接地極の向き**、スイッチの **ON/OFF 方向**を確認してから固定」
「**家具・エアコン位置**は最新版を見て、変更があれば止めて連絡を」
「1個ずつ**レベル当て→写真→仮固定→全確認**で進めてください」

## 事前チェック3つ
1. **高さ・左右離れ・向き・連用ルール**が、展開図＋仕様書＋現場墨で一致しているか
2. **施主変更（エアコン・家具・棚）の最新反映**と他職種干渉ゼロが確認できているか
3. **水平器でレベル取り→通電確認→写真**の段取りが組めているか

> 補足：高さの一般例（仕様書で要確認）：**コンセント FL+250〜300mm／スイッチ FL+1,100〜1,200mm**。
> 補足：壁面離れの目安：**ドア枠・コーナーから 100mm 以上**。家具・棚と被らない位置に。
> 補足：「**現場で確認しながら決める**」がプロ。図面を盲信せず、最新墨出しと施主変更を必ず照合。

## NG行動
- 高さを「いつもの位置」で済ませる
- 展開図を見ずに電気設備図だけで判断する
- 水平器を当てずに目分量で取り付ける
- 家具・エアコン位置の施主変更を確認しない
- ドア枠・コーナーからの離れを意識しない
- スイッチON/OFF方向・コンセント接地極を確認しない
- 連用枠の隙間・ガタつきを放置する
- 検電・通電確認を省略する

## 一言まとめ
スイッチコンセントは **「毎日使う顔」**。**高さ・離れ・向き**を仕様書通りに、**施主変更（家具・エアコン）と現場墨**を最新で照合。**水平器でレベル→通電→写真**で1個ずつ仕上げる。
