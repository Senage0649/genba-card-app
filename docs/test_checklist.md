## 確認手順の参考

### スマホ実機テストの流れ
1. Mac でターミナルを開き、`cd /Users/gyakusekawayuu/壁打ち` → `python3 -m http.server 8000`
2. 別ターミナルで `ipconfig getifaddr en0` を実行し、Mac の IP（例：`192.168.1.5`）を控える
3. スマホを Mac と同じ Wi-Fi に接続
4. スマホのブラウザで `http://(MacのIP):8000/app/home.html` を開く
5. このチェックリストを上から順にチェックしていく
