# 動作確認で誰を呼べばいいかわからない

## 今やること3つ
1. **連動表＋試運転マニュアル＋前工程OK記録**を揃える
2. **誰が・いつ・何を確認するか**を立会者全員で合意
3. **異常時の戻し手順＋写真・動画・測定値の記録**の段取り

## まず見る資料
- 動作仕様書・連動表・シーケンス図
- 単線図・分電盤図・制御盤詳細図
- 機器取説・メーカー図面・試運転マニュアル
- 特記仕様書・竣工検査要領
- 020〜023 の試験記録

## 誰に聞く？
- 元請け現場監督（最優先・全体調整）
- 自社の先輩・電気工事責任者
- 消防設備業者・消防設備士
- 機械設備業者・メーカー
- 弱電・火報・空調・防災業者
- 施主・管理会社・設計者

## 聞き方テンプレ
- 「動作確認の**立会スケジュール**を組みたいです、いつが空いていますか？」
- 「**連動表**で立会が必要な業者を割付したいです、確認お願いします」
- 「電気側は前工程OKです、**機械屋さんを呼ぶ音頭**を取りたいです」

## 職長への指示テンプレ
- 「動作確認は**前工程OK記録**を全部揃えてから」
- 「**単独 → 無負荷 → 連動無負荷 → 連動負荷**の順で進めてください」
- 「**写真・動画・測定値**を必ず残してください」
- 「異常出たら**即停止・連絡**、様子見禁止」

## 怒られるポイント
- 一人で勝手に動作確認 → 連動ミスで再工事、「なぜ連絡しなかった」
- 施主・メーカー不在で「正常動作した」 → 証拠不足で後日再確認命令
- 連動表を見ずに部分確認だけ → 連動先で動作不良
- 機械屋・メーカーを呼ぶタイミングを逃す → 工程後ろ倒し

## NG行動
- 一人で動作確認を完結させる
- 連動表を見ずに単独動作だけでOKと判断する
- 写真・動画・測定値を残さない

## 一言まとめ
**連動表で立会者を割付 → 単独 → 無負荷 → 連動無負荷 → 連動負荷 → 引渡し**。**音頭は電気屋が取る**。**証拠と立会**で「やった」を証明。
