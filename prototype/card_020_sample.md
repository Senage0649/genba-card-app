# 結線確認で何を見ればいいかわからない

## 今やること3つ
1. **単線図＋複線図＋取説**で結線の全体像と端子位置を把握
2. **無電圧確認 → 導通 → 端子/色/極性 → 締付**の手順を回す
3. **写真（Before/After）と他者目視確認**で証拠と二重チェック

## まず見る資料
- 単線図（系統図）
- 複線図・結線図・盤内詳細図
- 特記仕様書・設計変更指示書・質疑応答書
- 機器取扱説明書・メーカー図面
- 自社の結線標準（電線色・マーキングルール）

## 誰に聞く？
- 元請け現場監督
- 自社の先輩・施工管理者
- 自社職長（社内の結線標準・トルク値）
- メーカー（特殊機器・制御盤の端子）
- 弱電・火報業者（連動回路の責任分担）

## 聞き方テンプレ
- 「**単線図と複線図**、両方ありますか？複線図がないと終端の本数が読めません」
- 「この回路の**無電圧確認**、立ち会ってもらえますか？」
- 「**規定トルク**は仕様書のどこですか？」

## 職長への指示テンプレ
- 「結線前に必ず**ブレーカーOFF＋無電圧確認**を」
- 「**負荷側ショート→テスター導通**で1回路ずつ確認してください」
- 「制御線を外す時は **Before写真**を必ず、復旧用です」
- 「**17時以降の作業**は他者確認を必ず入れてください」

## 怒られるポイント
- 逆相・色間違い・緩みで機器が動かない／消防設備不動作
- 制御線を写真撮らずバラす → 復旧不能で大目玉
- 締め付け不足で接触不良 → 発熱・火災リスク
- 「説明したのに施工しない」 → 「甘やかさない」

## NG行動
- ブレーカーを落とさず結線する
- 検電（無電圧確認）を省略する
- 制御線を写真なしで外す

## 一言まとめ
**無電圧確認 → 負荷側ショート → テスター導通 → 端子/色/極性 → 締付 → 写真 → 他者目視**。**疲れた自分を信用せず**、単線図と複線図の両方で一筆書き。
